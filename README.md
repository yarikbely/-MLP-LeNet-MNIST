# Лабораторная работа №2: Сравнение MLP и LeNet на MNIST

## Результаты

### MLP (Scikit-learn)
- Точность: 97.85%
- Полнота: 97.85%
- F1-мера: 97.84%
- Время обучения: 45.2 сек

### LeNet (PyTorch)
- Точность: 99.05%
- Полнота: 99.05%
- F1-мера: 99.05%
- Время обучения: 168.5 сек

## Выводы

1. **LeNet превосходит MLP по точности** на 1.2%, что является значительным улучшением
2. **MLP обучается быстрее** в 3.7 раза (45 сек против 168 сек)
3. **LeNet имеет меньше параметров** (~60K против ~100-300K у MLP)
4. **Сверточные сети лучше подходят** для задач компьютерного зрения

## Итоговая рекомендация
- Для production-систем с требованием высокой точности → **LeNet**
- Для быстрого прототипирования → **MLP**
